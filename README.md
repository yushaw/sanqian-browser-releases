### Manual Installation

1. Download zip from the [Releases](https://github.com/yushaw/sanqian-browser-releases/releases) page
2. Unzip the file
3. Open Chrome and navigate to `chrome://extensions/`
4. Enable "Developer mode" (toggle in top-right corner)
5. Click "Load unpacked" and select the unzipped folder

## üöÄ Quick Start

### Prerequisites

Sanqian Browser requires a local backend service to function. You have two options:

**Option 1: Download Pre-built Binary** *(Recommended)*
- Visit [sanqian.io](https://sanqian.io) to download the client application
- Follow the installation instructions for your OS

**Option 2: Build from Source**
- Clone the main repository: `git clone https://github.com/yushaw/sanqian-browser`
- Follow the [setup instructions](https://github.com/yushaw/sanqian-browser#readme)

### Setup

1. **Start the backend service**
2. **Install the Chrome extension** (see Download section above)
3. **Configure API keys:**
   - Click the extension icon
   - Go to Settings
   - Add your LLM API key (OpenAI, Anthropic, Gemini, or Ollama)
4. **Start automating!**
   - Click the extension icon to open the side panel
   - Type your task in natural language
   - Let AI agents handle the rest
