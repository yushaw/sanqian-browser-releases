# Promotional Images for Chrome Web Store

This directory should contain promotional/marketing images for Chrome Web Store.

## Required Image

### Small Promo Tile (REQUIRED)
- **Size**: 440x280 pixels
- **Format**: PNG or JPG
- **Filename**: `small-promo-440x280.png`
- **Usage**: Shown in search results and featured sections

**This is REQUIRED for Chrome Web Store submission.**

## Optional Images (Recommended)

### Large Promo Tile
- **Size**: 920x680 pixels
- **Format**: PNG or JPG
- **Filename**: `large-promo-920x680.png`
- **Usage**: Featured placements, larger visibility

### Marquee Promo Tile
- **Size**: 1400x560 pixels
- **Format**: PNG or JPG
- **Filename**: `marquee-1400x560.png`
- **Usage**: Hero banner in featured sections

### Design Principles

1. **Keep it Simple**
   - Clean, uncluttered design
   - Easy to read at small sizes
   - Professional appearance

2. **Brand Colors**
   - Use consistent colors with your extension
   - Consider: Blue (trust), Green (go/action), Purple (innovation)
   - Avoid too many colors (2-3 max)

3. **Text Hierarchy**
   - Large: Extension name
   - Medium: Tagline
   - Small: Features/benefits

4. **Readability**
   - High contrast between text and background
   - Readable font (sans-serif recommended)
   - Not too much text

## Step-by-Step Creation (Canva)

1. **Sign up / Login** to Canva
2. Click **"Create a design"**
3. Enter **custom dimensions**: 440 x 280
4. Choose a template or start blank
5. **Add elements:**
   - Upload your icon (128x128)
   - Add text: "Sanqian Browser"
   - Add tagline: "AI-Powered Web Automation"
   - Add features if space allows
6. **Style:**
   - Choose background color or gradient
   - Adjust text colors for contrast
   - Align everything nicely
7. **Download:**
   - Click "Share" â†’ "Download"
   - Format: PNG (recommended)
   - Quality: High
8. **Repeat** for other sizes (920x680, 1400x560)

## Quality Checklist

## Tips for Success

1. **Test at Actual Size**
   - View your image at 100% zoom
   - Check if text is readable
   - Ensure details are visible

2. **Mobile-Friendly**
   - Small tile will be viewed on phones
   - Keep text large enough
   - Simple designs work better

3. **Brand Consistency**
   - Match your extension's color scheme
   - Use same icon/logo
   - Consistent typography

4. **Stand Out**
   - Look at competitor extensions
   - Find ways to differentiate
   - But stay professional
